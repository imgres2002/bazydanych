pwd - wyświelta w jakim katalogu się aktualnie znajdujemy
ls - wyświetla zawartoś danego katalogu
cd (nazwa katalogu)/ - przenosi nas do danego katalogu
cd .. - cofa nas do poprzedniego katalogu
clear - czyści konsole

mkdir (nazwa katalogu) - tworzy nowy katalog
rm -r (nazwa_katalogu) - usuwa dany katalog
touch (nazwa_pliku.rozszerzenie) - tworzy nowy plik
rm (nazwa_pliku.rozszerzenie) - usuwa dany plik

git status - wyświetla stan lokalnego repozytorium
git log - historia commit'ów
przyciskiem 'q' wychodzimy z menu
git branch - wyświetla listę branch'y które mamy w lokalnym repozytorium oraz wyświetla na którym się znajdujemy

git add - wybieramy plik które chcemy przenieść z WORKING DIRECOTORY(nasz komputer) do STAGING AREA(poczekalnia)
git add --all - dodaje wszystkie pliki


git commit - wybieramy pliki które chcemy przenieść z STAGING AREA(poczekalnia) do lokalnego repozytorium
git commit -m "wiadomość" - wysyła pliki wraz z wiadomością
git commit -am "wiadomość" - łączy komende git add z git commit -m (UWAGA nie przenosi nowo utworzonych plików tylko te które już były)

git push - wysyła plik z lokalnego repozytorium do właściwego repozytorium
